# Flight Xbee plot

Getting data via xbee, and it as a plot.

**Inspire from https://github.com/KuoE0/MPU6050-WebPlot**

## Start 

After plugin your xbee, enter `dmesg` to find your serialport, to start getting data enter:

```
sudo node xbee.js
```


## Develop

```
npm install
```
